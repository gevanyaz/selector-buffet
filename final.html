Selector Buffet
(CSS Selectors)


Simple Selectors

Universal Selector
* { property: value; }

Also Known as a wildcard, the universal selector selects every single element in the document.

Type Selector
p{ property: value; }

The type selector selects all HTML elements of the specified type (in the example it is represented as "p").

Class Selector
.my_class { property: value: }

The class selector matches and selects HTML elememts based on the value of the class that was given, selecting every single element in the document with that specific class name (in the example it is ".my_class").

ID Selector
#my_id { Property: value; }

The ID Selector selects an HTML element based on the valur of its ID attribute. Can only be used once in a document.

Simple Selector Examples

Attribute Selectors

Attribute Selector
[attribute] {property: value;}

The attribute selector is used by using a pair of square brackets,[], to selects the wanted attribute.

Attribute Value Selector
[attribute=value] {property: value;}

The value of the attribute comes after putting the "='value'" after the attribute itself.

Prefix Value Selector
[attribute^=value] {property: value;}

The caret,^, symbol is used as a prefix to select the elements based on the beginning of thier attribute values.

Suffix Value Selector
[attribute$=value] {property: value;}

The symbol $ is used as a suffix to select the elements based on the ending of the attribute values.

Attribute Value Substring Selector
[attribute*=value] {property: value;}

The value substring selector matches elements with an attr attribute, whose value begin with "value".

Attribute With Hyphen Separator Selector
[attribute|=value]{property: value;}

[lang|=en] The hyphen seperator selector is used to select elements with the attribute value followed by a hyphen (-)

Attribute With Multiple Values Selector
attribute~=value

Selects elements with an attribute value containing a specified word; when some elements have a list of space seperated attribute values.

Attribute Selectors Examples

<style> [target] { background-color: yellow; color: black; } [href="https://www.example.com"] { color: blue; } [class^="btn"] { font-weight: bold; } [src$=".png"] { border: 1px solid #ccc; } [title*="flower"] { color: green; } [lang|="en"] { font-style: italic; color: white; } [class~="important"] { color: red; } </style>

Combination Selectors

Descendent Selector
div p { font-size: 20px;}
-p descendents of div would have their fonts changed

Only the descendant elements (p) of the specified parent element (div) will be selected.

Direct Child Selector
h1 > p { font-size: 20px;}
-p elements that are direct children of h1 would have their fonts changed

The direct child elements (p) of the specified parent element (h1) will be selected. Other elements inside of h1 that have a (p) it wouldn't be selected.

General Sibling Selector
h2 ~ p { color: green;}
any sibling p that follows after the h2 will be green

This will select all p elements that come after and are siblings to a h2 element

Adjacent Sibling Selector
h2 + p { color: purple;}
The next p that is a direct sibling to h2 will be purple

This selects the (p) element that is a sibling too and follows immediately after h2

Or Selector, (grouping selectors)
div, h3, p { color: red;}
All div, h3, p elements would turn red

Otherwise known as a grouping selector, lets you selects multiple elements at once.

Pseudo-Class Selectors for position

first child
(element):first-child { property: value; }

Selects every chosen element that is the first child of its parent

last child
(element):last-child { property: value; }

Selects every chosen element that is the last child of its parent

nth child
a:nth-child(n) { property: value; }

Selects chosen elemetn that are the (nth) child of its parent; can be used for any number or parity

first of type
p:first-of-type { property: value; }

Selects the first child, of the chosen element, of its parent.

last of type
p:last-of-type { property: value; }

Selects the last child, of the chosen element, of its parent.

Pseudo-Element Selectors

First-line pseudo-element
(element)::first-line { property: value; }

The first line of a block level element will get selected

First-letter pseudo-element
(element)::first-letter { property: value; }

The first letter of a block level element will get selected

Before pseudo-element
(element)::before { property: value; }

This will insert content before the content of the selected element

After pseudo-element
(elememt)::after { property: value; }

This will insert content after the content of the selected element

Psuedo-Element Examples

Pseudo-class Selectors for Links

a:link Selector
a:link { property: value; }

Targets unvisited links

a:visited Selector
a:visited { property: value; }

Targets visited links

a:active Selector
a:active { property: value; }

Targets the activated link

a:hover Selector
a:hover { property: value; }

Targets the link while it is being hovered

a:focus Selector
a:focus { property: value; }

Targets the element that currently has focus; such as when it's tabbed to

Pseudo-class Selector for Input

input:focus Selector
input:focus { property: value; }

Selects an input elements that has focus

input:required Selector
input:required { property: value; }

Selects input elements based on whether they have the 'required' attribute

input:checked Selector
input:checked { property: value; }

Selects any radio or checkbox input is checked

input:disabled Selector
input:disabled { property: value; }

Selects an disabled input element
